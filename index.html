Note: Scraping may take 5-10 seconds per website. Vercel has a 60-second timeout, so keep your CSV to ~5 websites or less for best results.

How to use

  • Prepare a CSV file with websites in the first column
  • The first row should be a header (e.g., website,company_name)
  • Upload the file and click "Start Scraping"
  • The scraper will check each website for emails and addresses
  • Download your results with new columns added

Example CSV format

website,company_name
